# Figma to CSS

## Purpose
Translate design artifacts into maintainable coded styles and structure.

## Inputs
- Figma or Pencil design reference
- target component or screen
- existing token system if present

## Process
1. extract reusable visual rules
2. map to tokens before writing ad hoc values
3. convert layout intent into practical CSS structure
4. align states and variants with implementation needs

## Outputs
- token mapping
- CSS structure notes
- implementation-ready style guidance
