# TypeDB Query Author

## Purpose
Write or refine TypeDB queries that are readable and semantically correct.

## Inputs
- retrieval objective
- relevant schema types
- constraints and expected shape

## Process
1. state the retrieval intent
2. map the intent to entities, relations, and attributes
3. write the clearest query that satisfies the need
4. verify assumptions about multiplicity and ownership
5. describe output mapping needs

## Outputs
- query draft
- schema assumptions
- result-shaping notes
