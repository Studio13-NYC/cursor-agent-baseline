# Contract Diff

## Purpose
Compare current and proposed interfaces and identify downstream impact.

## Inputs
- old contract
- new contract
- affected consumers and producers

## Process
1. list the changed fields, behaviors, or guarantees
2. classify changes as additive, breaking, or internal
3. identify impacted agents and modules
4. note migration and validation requirements

## Outputs
- contract delta summary
- impact list
- migration notes
