# Prompt Template

## Purpose
Create a reusable prompt pattern with explicit inputs and outputs.

## Inputs
- task objective
- context format
- desired output shape
- guardrails

## Process
1. define the model's role narrowly
2. specify the inputs clearly
3. define the expected output format
4. include constraints and failure handling
5. keep wording stable enough to test

## Outputs
- prompt template
- variable list
- output contract
